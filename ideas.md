# Brainstorm de Design - Calculadora de Vigas de Concreto Aéreas

## Ideia 1: Engenharia Moderna Minimalista
**Design Movement:** Brutalism Digital + Modernismo Suíço
**Probability:** 0.08

### Core Principles
- Honestidade estrutural: mostrar a lógica dos cálculos visualmente
- Tipografia forte e hierarquia clara
- Uso estratégico de espaço em branco
- Cores neutras com acentos técnicos (azul-industrial, cinza-concreto)

### Color Philosophy
- **Paleta:** Cinza-concreto (#4A4A4A), Azul-industrial (#1E40AF), Branco puro (#FFFFFF), Laranja-alerta (#FF8C42)
- **Intenção:** Evocar confiança, solidez e profissionalismo. O cinza representa o concreto, azul a engenharia, laranja destaca valores críticos

### Layout Paradigm
- Grid assimétrico com painel lateral fixo para inputs
- Visualização 2D/3D em destaque central
- Cards de resultados fluindo verticalmente com progressão lógica
- Divisões claras entre entrada, cálculo e visualização

### Signature Elements
- Linhas diagonais sutis representando tensões estruturais
- Ícones geométricos customizados (viga, ferragem, carga)
- Indicadores visuais de "stress" (barras de progressão para capacidade)

### Interaction Philosophy
- Transições suaves ao alternar 2D/3D
- Hover states que revelam mais detalhes técnicos
- Animações que representam forças (setas de carga se movem sutilmente)

### Animation
- Entrada: elementos aparecem com fade + slide suave (200ms)
- Hover: aumento sutil de profundidade com shadow
- Transição 2D→3D: rotação suave com perspectiva
- Carregamento: barra de progresso com movimento linear

### Typography System
- **Display:** IBM Plex Sans Bold para títulos (títulos de seção)
- **Body:** IBM Plex Sans Regular para conteúdo (legibilidade técnica)
- **Mono:** IBM Plex Mono para valores numéricos (cálculos)

## Ideia 2: Construção Artesanal Intuitiva
**Design Movement:** Warm Modernism + Craft Aesthetic
**Probability:** 0.07

### Core Principles
- Acessibilidade para construtores sem formação técnica
- Visualização tangível e compreensível
- Uso de cores quentes e aconchegantes
- Diagramas e ilustrações customizadas

### Color Philosophy
- **Paleta:** Terra-queimada (#B8621B), Verde-madeira (#2D5016), Bege-areia (#F5E6D3), Vermelho-tijolo (#A0422E)
- **Intenção:** Criar sensação de conforto e familiaridade com materiais reais. Cores evocam obra, segurança e tradição construtiva

### Layout Paradigm
- Fluxo vertical narrativo (passo a passo)
- Cartões com bordas suaves e sombras quentes
- Ilustrações grandes e didáticas
- Seção de "dicas do construtor" integrada

### Signature Elements
- Ilustrações hand-drawn style (vigas, ferragens, operários)
- Badges com "tips" para construtores
- Ícones com estilo artesanal (não geométricos puros)

### Interaction Philosophy
- Cliques revelam explicações em linguagem simples
- Animações que "constroem" a viga gradualmente
- Feedback tátil visual (como se estivesse manipulando física)

### Animation
- Entrada: elementos "crescem" como se estivessem sendo construídos
- Hover: cards ganham sombra quente e se elevam
- Transição 2D→3D: rotação com efeito de "montagem"
- Resultado: números aparecem com efeito de "digitação"

### Typography System
- **Display:** Merriweather Bold para títulos (caloroso, confiável)
- **Body:** Lato Regular para explicações (amigável, legível)
- **Mono:** Courier Prime para valores (referência ao manual)

## Ideia 3: Dashboard Técnico Interativo Futurista
**Design Movement:** Cyberpunk Construtivo + Data Visualization
**Probability:** 0.06

### Core Principles
- Dados em primeiro plano com visualizações ricas
- Interatividade em tempo real (sliders, inputs dinâmicos)
- Estética tech-forward mas profissional
- Gráficos e simulações como protagonistas

### Color Philosophy
- **Paleta:** Roxo-neon (#7C3AED), Ciano-elétrico (#06B6D4), Preto-profundo (#0F172A), Amarelo-destaque (#FBBF24)
- **Intenção:** Criar sensação de controle, precisão e inovação. Cores neon transmitem tecnologia, enquanto mantém seriedade técnica

### Layout Paradigm
- Grid responsivo com widgets redimensionáveis
- Painel de controle esquerda, visualizações direita
- Gráficos e métricas em tempo real
- Modo "full-screen" para visualização 3D

### Signature Elements
- Gráficos com animações de dados (recharts com transições)
- Indicadores tipo "gauge" para capacidades
- Grid de fundo sutil (referência a interfaces futuristas)

### Interaction Philosophy
- Sliders que atualizam visualização em tempo real
- Cliques em elementos 3D destacam informações
- Modo comparativo (antes/depois de mudanças)

### Animation
- Entrada: elementos aparecem com efeito "scan" (linha horizontal)
- Hover: glow sutil ao redor de elementos interativos
- Transição 2D→3D: zoom com distorção de perspectiva
- Atualização de dados: números mudam com efeito de "flip"

### Typography System
- **Display:** Space Mono Bold para títulos (futurista, técnico)
- **Body:** Roboto Regular para conteúdo (limpo, moderno)
- **Mono:** JetBrains Mono para valores (profissional, código-like)

---

## Decisão Final
Será utilizada a **Ideia 1: Engenharia Moderna Minimalista** por ser a mais apropriada para construtores que precisam de confiabilidade, clareza técnica e profissionalismo. A abordagem minimalista reduz fricção cognitiva, enquanto a tipografia forte e cores técnicas reforçam credibilidade.
